AI产品经理5周特训

特训第 1 周: 四个维度快速补齐 AI 产品通识

本周目标

  1. 认知产品格局：了解 AI 产品的分类，有几大类，每一类的重要代表产品及其服务对象和用户需求。掌握各类产品的发展情况，确保在面对陌生产品时能够将其归类，不至于迷茫。
  2. 熟悉发展历程：从时间轴的角度，了解本轮 AI 发展的里程碑时刻及其背后的代表性产品。分析每个类别的代表性产品的发展过程（版本迭代历程、用户数据变化）。
  3. 了解行业格局：了解全球及国内的代表性 AI 公司，明晰它们如何分类、规模及发展历程，并掌握这些公司开放的岗位等信息。
  4. 弄懂技术格局：对支撑本轮 AI 浪潮的核心技术有全局性的认知。本周只需了解这些技术的名称和用途，后续会有专门的“AI 产品技术课”进行深入讲解。

特训第 2 周: Input & Output: AI 产品必备的技术基础（上） - LLM 篇

本周目标

  1. 熟知 LLM 技术格局： LLM 是生成式智能的骨干技术，是这波 AI 产品革命的核心技术基石。
  2. 深刻理解 LLM 的输入输出： 深入理解基于 LLM 的 Prompt Engineering、RAG、Agent 设计与技术。
  3. 掌握 Prompt Engineering 实战技巧： 掌握 8 种在工作中总结并升华的 Prompt Engineering 实战技巧。
  4. 深入阐述基于 LLM + Agent 技术的基本原理与技术细节。
  5. 解析代表性产品的实现原理与细节： 通过“伪代码”的方式，在文档中推演 LLM 代表性产品的基本框架（注意，作为产品经理，需要具备推演可实现性的能力，而非实际编写代码）。

特训第 3 周: Input & Output: AI 产品必备的技术基础（下） - 多模态篇

本周目标

  1. 熟悉各模态生成的技术格局： 掌握这波 Generative AI 中各个模态（图片、视频、音频、音乐）的生成技术格局。这些模态由 LLM 作为核心骨干，并以自然语言作为交互媒介，是 AI 产品经理必须掌握的技能包。
  2. 熟悉 text2image 的代表性技术原理。
  3. 掌握 LoRA 技术原理，并能上手制作 LoRA 模型。
  4. 熟悉语音技术原理，并能测试各大 text2speech 应用。
  5. 掌握 text2video 技术原理及其代表性产品（如 Pika、Runway、国内的 Pixverse 等）。
  6. 了解 text2music、歌声合成等其他方向的技术原理，作为补充。
  7. 深刻理解 LLM 如何调用这些技术： 确保在未来的 AI 产品岗位工作中，能灵活运用各项技术，在定义设计产品时游刃有余。

特训第 4 周: 扒产品！扒产品！扒产品！扒产品是产品经理最好的学习方法

本周目标

  1. 完成 3 份高质量的产品拆解调研报告。
  2. 深入理解产品拆解调研（俗称：扒产品）的方法论： 确保在未来需要拆解任何产品时能够轻松应对。
  3. 深入理解 AI 产品的行业格局： 明确哪些公司在做哪些产品，这些产品的发展情况及实现方式。确保在成为 AI 产品经理的第一时间就能直接上手（在新兴行业中，很多公司招聘时最关心的是面试者是否能够直接上手工作。这对于应对快速发展的 AIGC 行业尤其重要）。

特训第 5 周: 包装简历，规划求职路径，推荐岗位，选定目标公司

本周目标

  1. 获得一份超级吸引人的简历： 在简历中扬长避短，并在第一时间抓住招聘者的眼球。
  2. 收集足够多的职位描述（JD），并根据自身现状选择目标公司和岗位。
  3. 掌握产品博客的构建方法论： 构建简历之外的重要加分项——产品博客，并完成初步搭建，且具备长期维护的能力。这在应聘中，对于零基础的求职者来说，将是一个重要的加分项。
